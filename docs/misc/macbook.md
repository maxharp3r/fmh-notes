core apps

* install brew
* install oh my zsh
* install apps:

  ```sh
  # standard
  brew install keepingyouawake iterm2 stats rectangle glances
  ```

  ```sh
  git config --global user.name "(name)"
  git config --global user.email (email)
  cat ~/.gitconfig
  ```
